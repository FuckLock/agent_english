# product-spec-builder 区分学习习惯与产品边界

**问题描述**：在英语学习项目需求收集中，用户提到平时喜欢看英语电影或短视频学习，AI 误把它收窄成“基于电影/短视频片段的口语陪练工具”，忽略了用户真正想做的是进一步系统学习英语的项目。

**触发场景**：用户纠正：“不是的，上面只是我平时学习英语方式……我中文精通，英语你肯定也是需要翻译的，翻译成中文，要不然我不懂什么意思。”

**教训/建议**：product-spec-builder 遇到用户描述“平时怎么学”时，应区分输入偏好、学习习惯、产品边界三件事；在语言学习类产品中，还应主动确认母语解释/翻译/双语对照是否是核心能力。
